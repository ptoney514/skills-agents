# Template 3: Simplified/Internal Format

This template is for quick internal postings, departmental announcements, or when a more streamlined format is needed.

## Structure

### Header
```
**[Position Title]**
[Department/Unit]
[Location] | [Full-time/Part-time] | [Employment Type]
```

### Position Summary (1-2 paragraphs)
Brief overview including:
- What the position does
- Who they report to
- Primary purpose/impact
- Key working relationships

### Responsibilities
Either:
- **Bulleted list** of 5-8 key duties
- **Grouped sections** with subheadings if responsibilities are diverse

### Qualifications

**Required:**
- Bulleted list of must-haves

**Preferred:**
- Bulleted list of nice-to-haves

### Additional Information
Include as relevant:
- Work schedule/flexibility
- Physical requirements
- Travel requirements
- Salary range (if disclosed)
- Benefits summary

### How to Apply
Simple, direct instructions:
"To apply, please submit [requirements] to [contact/link] by [deadline]."

Or: "Interested candidates should contact [name] at [email/phone]."

## Tone Guidelines

- Clear and straightforward
- Less formal than external postings
- Focus on practical details
- Assumes familiarity with the organization
- Can be more conversational

## When to Use

- Internal job postings
- Quick-turnaround announcements
- Department-specific roles
- Contract or temporary positions
- When a full formal posting isn't needed
- Community bulletins or newsletters
