# Prompt Template: Employee Communications

Use this template when you need to draft announcements, emails, or internal communications to employees.

## Template Structure

```
You are a [COMMUNICATIONS ROLE] drafting a [COMMUNICATION TYPE] for [AUDIENCE] at [ORGANIZATION].

PURPOSE:
[What this communication needs to accomplish - inform, persuade, comfort, celebrate, etc.]

CONTEXT:
- Organization: [Company name, industry, culture notes]
- Situation/Background: [What led to this communication?]
- Audience: [Specific employee group(s) and their characteristics]
- Timing: [When will this be sent? Why now?]
- Related Context: [Any recent events or changes that affect this message]

KEY MESSAGES (in priority order):
1. [Most important message - lead with this]
2. [Second message]
3. [Third message]
4. [Additional messages]

REQUIRED ELEMENTS:
- [Element 1 - e.g., "Clear action item with deadline"]
- [Element 2 - e.g., "Link to FAQ document"]
- [Element 3 - e.g., "Contact information for questions"]
- [Element 4 - e.g., "Reassurance about X concern"]

TONE & STYLE:
- [Primary tone - e.g., "Professional yet warm"]
- [Secondary quality - e.g., "Action-oriented"]
- [Cultural fit - e.g., "Consistent with our startup culture"]
- [Emotional tone - e.g., "Optimistic" or "Empathetic" or "Urgent"]
- [Formality level - e.g., "Business casual" or "Formal"]

FORMAT:
- Communication type: [Email / Intranet post / Memo / Announcement]
- Length: [Word count - e.g., "200-300 words" or "One page"]
- Structure: [Paragraphs / Bullets / Sections]
- Visual elements: [Headers / Callout boxes / Bold text / Links]
- Subject line: [If email, include subject line requirements]

AUDIENCE CONSIDERATIONS:
- Knowledge level: [What do they already know?]
- Likely concerns: [What are they worried about?]
- What they need from this: [Information? Action? Reassurance?]
- Communication preferences: [How do they like to receive info?]

AVOID:
- [Thing 1 to avoid - e.g., "Corporate jargon"]
- [Thing 2 to avoid - e.g., "Passive voice"]
- [Thing 3 to avoid - e.g., "Making it sound like homework"]
- [Thing 4 to avoid - e.g., "Overwhelming with detail"]

CALL TO ACTION:
- Primary action: [What do you want recipients to do first?]
- Secondary actions: [What else might they do?]
- Deadline: [If applicable]
- Resources: [Where can they get more information?]

BONUS ELEMENTS TO CONSIDER:
- [Optional element 1 - e.g., "Quick summary box at top"]
- [Optional element 2 - e.g., "FAQ teaser"]
- [Optional element 3 - e.g., "Personal story or example"]
```

## Communication Types & Patterns

### Announcement Pattern:
```
Opening: What's happening
Why: Rationale/context
Impact: What it means for employees  
Action: What they need to do
Resources: Where to learn more
```

### Policy Change Pattern:
```
Opening: What's changing
When: Effective date
Why: Business rationale
Impact: How it affects different groups
Support: Resources and accommodations
Questions: FAQ and contacts
```

### Crisis Communication Pattern:
```
Opening: What happened (facts only)
Status: Current situation
Safety: Any immediate concerns addressed
Next steps: What we're doing
Support: Resources available
Updates: When to expect more information
```

### Celebration Pattern:
```
Opening: What we're celebrating
Recognition: Who contributed/achieved
Impact: Why it matters
Appreciation: Thank you message
Looking forward: What's next
```

## Common Additions

### For Complex Changes:
Add to REQUIRED ELEMENTS:
```
- Timeline of implementation phases
- Decision tree or flowchart for different scenarios
- Comparison table (current vs. new)
- FAQ document link
```

### For Action-Required Communications:
Add to CALL TO ACTION section:
```
- Step-by-step instructions
- Screenshot or visual guide
- Help desk information
- "What if I don't complete by deadline?" section
```

### For Sensitive Topics:
Add to TONE & STYLE:
```
- Empathetic and supportive
- Acknowledge difficulty or frustration
- Avoid dismissive or minimizing language
- Provide psychological safety
```

### For Distributed Workforces:
Add to AUDIENCE CONSIDERATIONS:
```
- Time zone considerations
- Access to technology
- Language/translation needs
- Cultural sensitivities
```

## Length Guidelines by Type

**Quick Updates:** 50-100 words
- Single fact or reminder
- No action required or very simple action

**Standard Announcements:** 200-300 words
- Multiple messages
- Clear action items
- Brief context

**Complex Communications:** 400-500 words
- Significant changes
- Multiple stakeholder groups
- Detailed explanations needed

**Crisis Communications:** 150-250 words
- Facts only
- Clear and direct
- Reassurance and resources
